ocaml-browser-vm

Static data for the in-browser dune terminal of the NPTEL course Functional Programming with OCaml: a bootable Linux VM image (OCaml 5.4 + dune + bisect_ppx) served as a boot snapshot plus a lazily fetched chunk store to the v86 emulator running in your browser.

Nothing executes on this server; the VM runs entirely in the visitor's tab on the course's playground page.
